# BF-006 — Translate Entire Document

## Goal

Translate every Translation Unit in the Document.

---

## Responsibilities

Implement:

- Iterate over every Translation Unit.
- Load the translation prompt once.
- Translate every unit using the existing OpenAI client.
- Preserve ordering.
- Return an ordered collection of TranslationResults.

---

## Out of Scope

- HTML reconstruction
- EPUB
- Cache
- Retry
- Parallel execution

---

## Acceptance Criteria

Every Translation Unit is translated.

Ordering is preserved.

No new infrastructure is introduced.
