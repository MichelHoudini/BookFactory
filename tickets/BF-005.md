# BF-005 — First Translation

## Goal

Translate a single Translation Unit using the Prompt Registry and the OpenAI client.

This milestone validates the complete path from the document model to the LLM.

---

## Responsibilities

Implement:

- Load the translation prompt.
- Select the first Translation Unit.
- Build the input text from its Blocks.
- Call the OpenAI client.
- Print the translated text.

---

## Out of Scope

Do NOT implement:

- Cache
- Retry
- EPUB
- Reconstruction
- Multiple Translation Units
- Parallelism
- Cost tracking

---

## Acceptance Criteria

Running the application with a valid API key produces a translated Translation Unit in the terminal.

The implementation reuses existing modules without duplicating logic.

No new infrastructure is introduced.
