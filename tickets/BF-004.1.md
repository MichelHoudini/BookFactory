# BF-004.1 — Standard `.env` Support

## Goal

Add standard `.env` support to application settings.

---

## Requirements

- Add `python-dotenv` as a runtime dependency.
- Load the `.env` file exactly once inside `src/bookfactory/config/settings.py`.
- Keep `Settings` as the only module allowed to access `os.environ`.
- Do not change any public APIs.
- Do not modify application behavior.
- Do not add any new infrastructure.

---

## Acceptance Criteria

- Values from `.env` are available through `Settings.from_environment()`.
- Existing environment variables retain standard precedence over `.env` values.
- `.env` loading occurs exactly once for the lifetime of the process.
- No module outside `src/bookfactory/config/settings.py` accesses `os.environ`.
- The complete test suite passes.
- Strict `mypy` passes.
- `git diff --check` passes.
