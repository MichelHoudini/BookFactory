# BF-004 — OpenAI Client

## Goal

Implement the first LLM client for Book Factory.

The implementation must communicate with OpenAI using the official OpenAI SDK while remaining replaceable by future providers.

---

## Responsibilities

Implement:

- Define an LLMClient contract.
- Implement OpenAIClient.
- Read the API key from environment variables.
- Read the model name from environment variables.
- Accept:
    - prompt
    - input text
- Return plain text.
- Raise clear domain-specific exceptions.

---

## Default configuration

OPENAI_MODEL=gpt-4.1-mini

---

## Out of Scope

Do NOT implement:

- Translation pipeline
- Retry
- Cache
- Cost tracking
- Streaming
- Multiple providers
- JSON mode
- Batch requests

---

## Acceptance Criteria

A single request can be successfully sent to OpenAI.

The response is returned as plain text.

Changing the model requires only changing an environment variable.

The implementation follows PROJECT.md and docs/ARCHITECTURE.md.
