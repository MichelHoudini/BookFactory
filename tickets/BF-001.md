# BF-001 — HTML Reader

## Goal

Implement the first executable version of Book Factory.

The application must read a single HTML document from the input folder and build the canonical in-memory Document model.

This milestone exists only to validate the document ingestion architecture.

---

## Inputs

A single HTML file located in:

input/

---

## Responsibilities

Implement:

- Read one HTML file.
- Parse the HTML.
- Extract the document title.
- Extract the author if present.
- Extract headings (h1–h6).
- Extract paragraphs in reading order.
- Build the canonical Document object.
- Print a readable summary to the terminal.

---

## Example Output

Book Factory

Book:
City of the Dog

Author:
John Langan

Headings:
17

324

Status:
Ready

---

## Out of Scope

Do NOT implement:

- Translation
- Chunking
- Providers
- Prompt loading
- Cache
- Validation
- EPUB generation
- Configuration
- Logging
- Retry
- Pipeline engine

---

## Acceptance Criteria

The project runs successfully with:

python src/app.py input/book.html

The application exits successfully.

The extracted information is correct.

The implementation follows PROJECT.md and docs/ARCHITECTURE.md.

No future milestones are anticipated.

Only the minimum architecture necessary for BF-001 should be implemented.
